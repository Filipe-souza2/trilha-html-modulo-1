História dos computadores

O primeiro computador foi o Mark 1 em 1944, um computador eletromecânico que usava válvulas. Em 1946, também tivemos o ENIAC, na época da Segunda Guerra Mundial. Foi nessa época do ENIAC que surgiu o termo "BUG". Em seguida, tivemos a segunda geração, que foi de 1959 a 1965, onde houve a evolução das válvulas para os transistores. Na terceira geração, de 1965 a 1970, os transistores evoluíram para os circuitos integrados. Hoje estamos na quarta geração, que começou em 1971.

O que são Clients?

O cliente é o dispositivo que está consumindo uma informação na internet. O cliente se conecta com outro computador pela internet; esse outro computador é o servidor.

O que são Servers?

Os servidores servem o cliente com as informações que estão armazenadas dentro deles, dependendo do que o cliente requisitar. Existem servidores na internet, que são os que estão na web, e intranets, que são servidores locais. Existem servidores de: arquivos, segurança (firewall), streaming, e-mail, web.

Como é uma estrutura básica de um HTML?

A estrutura basica do HTML é a tag html que delimita o inicio e o fim do html em si, temos uma tag head que indica oque será carregado, na tag body temos tudo que sera visto pelo cliente dentro da tag body fica tudo que vai ser visto no site é o corpo do site, toda tag tem o seguinte corpo tag aberta: '< elemento >' conteudo da tag, fechamento da tag: '< / elemento >', dentro dessas tags principais podemos usar diversas calsses aqui vão algumas delas:

  • Tag i deixa o texto em italico;
  • Tag strong deixa o texto em negrito;
  • Tag input cria um campo ;
  • Tag p cria um paragrafo;
  • Tag h1 ate h6 criam cabeçalhos;
  • Tag u é usada para deixar o texto sublinhado;
  • Tag a utilizamos para fazer links ;
  • Tag mark usamos para destar uma palavra ou um texto como um marca texto;
  • Tag smal é utilizada para fazer um texto menor;
  • Tag hr é usada para fazer uma divisão no seus conteudos com uma linha;
  • As Tags sub e sup são usadas para mover para cima e para baixo o texto;
  • Tag blockquote é usada para fazer uma citação;
  • As tags ul,ol,li são usadas para lista com ordenação e sem ordenação;
  • Tag font é usada para mudar aparencia do como fonte cor e tamanho, no entanto é uam tag que foi retirada no html 5 e não é mais usado assim a tornando obsoleta;
  • Tag del representa uma parte do texto que foi excluida ele faz um risco no meio da texto;
  • Tag abbr eu uso para fazer uma abreviação LOL.

Entendendo a diferença entre listas ordenadas e não ordenadas

Ambas as tags são utilizadas para fazer listas ol usada para lista com ordenação e ul lista sem ordenação li é uma tag filha que herda as propriedades da tag e pai e usada para que possa preencher a lista.

Exemplo de lista ordenada:

  1. 1
  2. 2
  3. 3

Exemplo de lista sem ordenação:

  • 1
  • 2
  • 3
